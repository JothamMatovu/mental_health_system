Mental Health Management System

This repository contains a PHP + MySQL web application. GitHub Pages cannot run PHP, so the full system will not work on this URL.

Run locally with XAMPP:

1) Start Apache + MySQL

2) Open setup: http://localhost/mental_health_system/setup/setup.php

3) Open app: http://localhost/mental_health_system/

If you want to deploy online, use a PHP-capable host (cPanel/shared hosting) or a VPS with Apache/Nginx + MySQL.
